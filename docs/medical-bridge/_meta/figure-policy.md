# Figure Policy · 真实图优先

Peter 明确要求：CNS / 高影响案例深读里多放真实结果图，不要只用纯汉字流程图。

## 优先级

1. Wikimedia Commons、NCI、FDA、CDC、NIH、WHO 等可公开使用图。
2. 开放获取论文中许可允许复用的 Figure。
3. 版权不便直接复用的高影响论文：放“真实 Figure 入口”，指向原文 Figure 页面，并在笔记里逐图拆解。
4. Mermaid：只用于机制框架、因果链、决策树，不能替代真实实验结果图。

## 每张图必须回答

- 这张图的观测单位是什么：病人、样本、细胞、spot、基因、克隆还是时间点？
- 统计动作是什么：分组比较、回归、聚类、轨迹、空间共定位还是生存模型？
- 生物学声明是什么：机制、标志物、分型、耐药、预后还是治疗反应？
- 结论边界是什么：强支持、弱暗示、不可外推分别在哪里？

## 图注最低要求

- 来源链接。
- 作者 / 机构和许可。
- 是否改动。
- 访问日期。
