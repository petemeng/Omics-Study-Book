# L5 · 课题设计模式库

> L5 是客户咨询时能立刻调用的 playbook。每篇都要能从“医生一句话”落到“样本、组学、分析、风险、产出”。

## 标准课题模式

- 治疗反应分子机制研究。
- 原发耐药 vs 获得性耐药。
- 预后分子标志物筛选。
- 早期诊断标志物筛选。
- 疾病分子分型重构。
- 罕见亚型 / 罕见疾病探索。
- 治疗靶点发现。
- 微环境与免疫互作。
- 疾病进展与转移。
- 药物联用机制。
- 真实世界证据 + 多组学验证。
- 基础研究的临床转化设计。

## 客户类型

- 资深 PI 型医生。
- 临床研究型医生。
- 转化兴趣型医生。
- 第一次做组学的医生。
- 有失败经验的医生。
- 时间紧任务重的医生。

## 失败模式

- 取样设计致命错误。
- 样本量不足以回答主问题。
- 混杂因素未控制。
- 临床异质性被忽略。
- 选错组学。
- 数据出来后才设计分析。
- 解读过度引申。
- 转化路径断裂。
