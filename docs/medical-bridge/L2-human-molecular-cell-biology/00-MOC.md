# L2 · 人类分子细胞生物学

> L2 是 Peter 最需要重打的地基：不是背通路名，而是能解释“细胞为什么这样决策，疾病如何劫持它，临床药物打在哪”。

## 核心信号通路

- [[medical-bridge/L2-human-molecular-cell-biology/_pathways/pi3k-akt-mtor-core-map]] — 生长、代谢、存活和内分泌治疗耐药的核心轴
- RAS-RAF-MEK-ERK — 生长因子到核内转录程序
- p53-MDM2-p21 — DNA 损伤后的停周期、修复与凋亡选择
- CDK-Cyclin-Rb-E2F — 细胞周期闸门
- BCL-2 family apoptosis — 线粒体凋亡开关
- ATM/ATR-CHK1/2 — DNA 损伤检查点
- JAK-STAT/IFN — 炎症、免疫和肿瘤免疫治疗
- NF-kB — 炎症、生存和免疫微环境

## 核心蛋白家族

- RTKs：EGFR、HER2、MET、FGFR、VEGFR、ALK、ROS1、RET、TRK。
- 非受体激酶：SRC、JAK、BTK、SYK。
- GTPase：RAS、Rho、Rab、Arf。
- 转录因子：MYC、E2F、NF-kB、STAT、HIF、SOX、FOX。
- 染色质修饰酶：DNMT、TET、HDAC、HAT、PRC2、SWI/SNF。
- E3 泛素连接酶：MDM2、VHL、SCF、APC/C、FBXW7。
- CD 分子：肿瘤免疫常用 marker 优先。
