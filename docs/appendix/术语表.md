# 术语表

| 术语 | 含义 |
|---|---|
| barcode | 用于标记样本、细胞或空间位置的短序列标签 |
| UMI | Unique Molecular Identifier，用于标记原始分子并减少 PCR 扩增偏差 |
| counts | reads 或 UMI 汇总后的计数矩阵 |
| TPM | Transcripts Per Million，常用于表达量展示 |
| peak | ATAC-seq 或 ChIP-seq 中 reads 富集的基因组区域 |
| motif | 转录因子偏好结合的短 DNA 序列模式 |
| DMR | Differentially Methylated Region，差异甲基化区域 |
| ASV | Amplicon Sequence Variant，扩增子分析中精确序列变体 |
| clonotype | 免疫组库中按 CDR3、V/J 基因等规则定义的克隆型 |
| LD | Linkage Disequilibrium，连锁不平衡 |
| eQTL | 影响基因表达水平的遗传位点 |
| pseudo-bulk | 将同一样本同一细胞类型的单细胞 counts 汇总后做样本级分析 |
| batch effect | 与生物问题无关但系统影响数据的批次差异 |
| confounder | 同时影响分组和结果、可能造成假关联的混杂因素 |
| FDR | False Discovery Rate，多重检验中的假发现率 |
| alpha diversity | 单个样本内部的微生物多样性 |
| beta diversity | 样本之间的微生物群落差异 |
| PRS | Polygenic Risk Score，多基因风险评分 |
| colocalization | 判断 GWAS 与 QTL 信号是否可能共享同一因果变异 |
